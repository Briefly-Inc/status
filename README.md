## ‚≠ê About Upptime

This repository uses [Upptime](https://upptime.js.org) - an open-source uptime monitor and status page powered by GitHub Actions and Pages.

### How it works

- **GitHub Actions** runs automated checks every 5 minutes to verify all services are online
- **Response times** are recorded and committed to git for historical tracking
- **GitHub Issues** are automatically opened when downtime is detected and closed when services recover
- **GitHub Pages** hosts the beautiful, accessible status website

For more information, visit the [Upptime documentation](https://upptime.js.org/docs).
